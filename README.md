## サービス概要
**カロリー摂取だけに特化したカロリー記録アプリ**

「Calorie_Bank」は、カロリー管理を楽しい習慣に変えるアプリです。
カロリー管理を「制限」ではなく「選択」というポジティブ思考に変えながら、カロリーを身近に感じる記録習慣をサポートします。摂取カロリーを抑えるのではなく、1日の消費カロリー内に収めた分を「貯金」として累積表示することで、ポジティブな達成感を提供します。完璧を求めず、まずはカロリーだけに特化して「習慣化する」ことを優先したアプリです。


## このサービスへの思い・作りたい理由

私自身、白米が大好きなのに「ダイエット＝大好きな白米を減らし、それでも食べる量を制限する」という、何もかも諦めるような感覚に苦しんだ経験があります。

ダイエット中に食事記録アプリを使っていましたが、細かく記録することが段々負担になり、食事のたびにプレッシャーを感じて続きませんでした。一方で、カロリーを調べていくうちに、食事は「我慢」ではなく「選び方」だと気づきました。同じカロリーでも工夫次第で満足度は変わる。そう思えるようになってから、食事を前向きに楽しめるようになりました。

また、プログラミング学習で「ハードルを下げて達成感を感じることで無理なく継続できる」という経験をしたことも、このアプリ開発のきっかけになりました。「1日の消費カロリー内に収めた分を貯金する」という発想の転換で、同じ行為でも心理的な受け止め方が全く変わると考えました。

本格的なダイエットというより、まずは気軽にカロリーを記録して始めてみたい方、白米を我慢せずに痩せたい方に、このサービスを提供したいと思いました。
## ユーザー層について

**ユーザー層：気軽にダイエットを始めたい人、白米が好きな人**

- 複雑な食事記録アプリで挫折した経験があり、「シンプルに続けたい」と思っている人
- 白米を食べることが好きで、我慢ではなく選び方で満足度を高めたい人
- 完璧な記録よりも、続けることを優先したい人
- 厳密な数値管理ではなく、おおよその目安で十分と考えている人



## サービスの利用イメージ

ユーザーは日々の食事のカロリーを記録します。栄養素や詳細な料理名は不要で、カロリーだけに特化したシンプルな記録です。MVP時は手入力でカロリーだけを記録し、本リリース後は画像認識機能も追加することで、より気軽に記録できるようになります。

記録したカロリーと1日の消費カロリーを比較し、消費カロリー内に収めた分を「貯金額」として可視化します。例えば、消費カロリーが2,000kcalで摂取カロリーが1,800kcalなら、200kcalの貯金成功です。貯金確認画面では、通帳のように時系列で記録が並び、累計貯金額が一目でわかります。

チートデイなどで食べ過ぎた際は「カロリーローン機能」（本リリース）で無理なく調整できます。オーバーした分を1週間または2週間で返済する設定にすることで、挫折せずに続けられます。

また、アンケート機能（本リリース）を通じて他のユーザーとつながり、気になることを質問・共有できます。一人で頑張るのではなく、コミュニティ全体で支え合いながらカロリー記録を続けられます。

記録を続けることでカロリーへの感覚が養われ、「だいたいこのくらい」がわかるようになり、食事選びが自然と楽しくなっていきます。

また、技術的に可能であれば、LINE Bot連携（本リリース）も実装予定です。LINEから手軽にメッセージを送るだけで記録できるようになり、アプリを開く手間なく習慣化をさらにサポートします。

## サービスの差別化ポイント・推しポイント

### 既存サービスとの違い

既存のダイエットアプリは「何をどれだけ食べたか」を詳細に記録します。栄養素、タンパク質、脂質、糖質...。本格的にダイエットしたい人には最適ですが、初心者には記録のハードルが高く感じられることがあります。

calorie_Bankは真逆のアプローチです。**記録するのはカロリーだけ。節制できたカロリーを「貯金」として見せる。** まずは気軽に始めたい人のための、入口となるアプリです。

### このアプリが解決する本質的な問題

ダイエットを続ける難しさの一つは、**心理的な負担が大きい**事だと考えます。
Calorie_Bankは、これらの心理的負担を軽減するようなアプローチを取ります。


**1. 貯金という言葉の力**
「節制した」「我慢した」→「貯金できた」 →　ポジティブ思考

**2. ローン機能という逃げ道**
完璧主義がダイエットを難しくします。「1回でも失敗したら終わり」じゃない。食べ過ぎても調整できる。この安心感が、長く続けられる秘訣です。

**3. カロリーだけに絞る勇気**
多機能なアプリは素晴らしいですが、初心者にはハードルが高いと考えます。Calorie_Bankは「まず始める」ことを最優先。習慣化できてから、必要なら他のアプリに移ればいい。その入口になることを目指しています。

### Calorie_Bankが提供する価値

食事記録ツールではありません。**習慣化するための、サポートツール**です。

白米を我慢せず、自分で選ぶ楽しさを感じながら、無理なく続けられる。そんな体験を届けたいと考えています。
