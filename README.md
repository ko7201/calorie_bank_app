## サービス概要
**食事を"楽しむこと"と体を"気づかうこと"を両立できるアプリ**

「calorie_bank」は、カロリー管理を楽しい習慣に変えるアプリです。

カロリーを記録することへのハードルを限りなく下げて、カロリーを身近に感じることを目指しています。摂取カロリーを抑えるのではなく、摂取カロリーより消費カロリーが上回った分を「貯金」として累積表示することで、ポジティブな達成感を重視しています。継続のハードルを下げ、完璧を求めず「意識を変える」「習慣化する」ことを優先したアプリとなっています。

## このサービスへの思い・作りたい理由
ダイエット中に食事記録アプリを使っていましたが、細かく記録することが段々負担になり、食事のたびにプレッシャーを感じて続きませんでした。一方で、カロリーを調べるうちに「好きな食べ物でも、これ1つ食べるカロリーで別のものなら3つ食べられる」とわかると、自然と「じゃあ3つ食べられる方にしよう」と思えるようになりました。これは「我慢」ではなく「選択」で、それを機にダイエット中の食事でも楽しくなりました。このような背景とプログラミング学習で学んだ「ハードルを下げて達成感を感じることで無理なく継続できる」という経験から、このアプリを作りたいと思いました。

## サービスの利用イメージ

**1. 目標設定**

初回登録時に、年齢・性別・身長・体重・活動量を入力すると、1日の消費カロリーが自動計算されます。目標体重を設定すると、必要な貯金額(例: 5kg減量 = 35,000kcal)が表示されます。

**2. 食事記録**

その日に食べたもののカロリーを登録していきます。

**3. 貯金が見える**

その日に節約できたカロリーが「貯金」として累積表示され、目標までの進捗が一目でわかります。美味しいものをたくさん楽しんだ日で摂取カロリーが消費カロリーを上回った日があっても、累計の貯金額が見えるため、罪悪感を感じにくく前向きに継続できます。

**得られる価値**
- カロリー管理が「苦痛」から「楽しみ」に変わる
- 「我慢」ではなく「選択」で食事を楽しめる
- 達成感を感じながら、無理なく継続できる

## サービスの差別化ポイント・推しポイント
一般的な食事管理アプリは「食べたものの記録」が中心ですが、
本サービスは 節制したカロリー＝貯金 として蓄積する仕組みを採用しています。
数字を追うだけではなく、「貯まっていく体験」で継続しやすい点が特徴です。
カロリーを「制限するもの」ではなく「貯めるもの」として捉え直すことで、ポジティブな達成感を重視しています。累積貯金額が常に見えるため、一時的に摂取カロリーが多い日があっても、全体の進捗が把握でき前向きに継続できます。

また、カロリーローン機能により特別な日にも柔軟に対応でき、「我慢」ではなく「計画的な選択」でダイエットを楽しめる点が最大の差別化ポイントです。


サービスの差別化ポイント・推しポイント

一般的な食事管理アプリが「食べたものの記録」を中心にしているのに対し、
本サービスは 節制したカロリー＝貯金 として蓄積する仕組みを採用しています。
カロリーを「制限するもの」ではなく “貯めていく資産” として捉えることで、
数字を追うだけの管理では得られない前向きな達成感を提供します。

また、貯まったカロリーが常に可視化されるため、
一時的に摂取カロリーが多い日があっても全体の進捗が把握でき、
モチベーションを維持しながら継続しやすい設計になっています。

さらに、カロリーローン機能を備えており、
飲み会など特別な日の「食べすぎ」を、後日計画的に調整できます。
「我慢」や「失敗感」を排除し、計画的に楽しむダイエットを実現する点が大きな差別化ポイントです。

このアプリで実現している価値

	•	カロリー管理を「貯金する体験」に変換
	•	達成感を重視したポジティブな継続設計
	•	一時的な食べすぎも管理できる柔軟性
	•	ローン機能による計画的なリカバリー
	•	継続ハードルを下げる“心理的な使いやすさ”
