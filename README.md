## サービス概要
**食事を"楽しむこと"と体を"気づかうこと"を両立できるアプリ**

「calorie_bank」は、ダイエットが長続きしない人でも楽しく健康になれるカロリー管理アプリです。

### 特徴
- **達成感を感じられる**: 摂取カロリーより消費カロリーが上回った分を「貯金」として見える化
- **楽しく続けられる**: 「制限」ではなく「貯金」というポジティブなイメージで継続をサポート
- **簡単に記録できる**:基本の食材や調味料が最初から表示され、ユーザーは量を記入するだけでOK(ユーザーが表示項目を編集することも可能)
### このアプリのコンセプト
食事記録が続かない人も、最初から完璧を求めず、**「意識を変える」「達成感を感じる」ことを最優先**にしたアプリとなっています。

## このサービスへの思い・作りたい理由
### ダイエットアプリが続かなかった経験
ダイエット中に食事記録アプリを使っていましたが、食材や調味料など細かく記録できることが最初は魅力的でしたが、段々と記録がしんどくなり、制限されている気分が強くなって続きませんでした。食事のたびに摂取カロリーがプレッシャーになり、達成感を得ることが難しかったのです。
### 意外な発見
しかし、カロリーを気にしていると意外な発見もありました。自分が好きな食べ物の中に、低カロリーであまり気にせず食べられるものがあったのです。そこから色々な食べ物のカロリーを調べることが、ゲーム感覚のように楽しくなりました。
### プログラミング学習からの学び
プログラミング学習で学んだことは、ハードルを高くしすぎず、タスクを小さくして"達成感"を感じることが"無理なく継続できる"という部分です。
この「継続するための工夫」をアプリにも活かせないかと考えました。
